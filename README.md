# 📊 CRM System - Gestão e Fluxo de Negócios

> Aplicação para gerenciamento de relacionamento com clientes (CRM), controle de pipeline de vendas, cadastro de contatos e acompanhamento de métricas operacionais.

---

## 🚀 Sobre o Projeto

Este projeto foi construído para resolver problemas reais de gestão de leads e processos comerciais. Unindo minha experiência prática em rotinas operacionais e análise de risco com o desenvolvimento de software, criei uma interface intuitiva para acompanhamento de clientes de ponta a ponta.

A aplicação é agnóstica de nicho, podendo ser utilizada por empresas de serviços, consultorias, escritórios ou e-commerce.

---

## 🛠️ Tecnologias Utilizadas

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Estilização:** CSS Flexbox / Grid
- **Controle de Versão:** Git e GitHub

---

## 🔥 Funcionalidades do Sistema

- [x] **Gestão de Contatos (CRUD):** Cadastro, edição e acompanhamento de clientes.
- [x] **Funil de Vendas (Pipeline):** Organização das negociações por etapas do processo comercial.
- [x] **Filtros e Buscas:** Localização rápida por nome, e-mail ou status.
- [x] **Histórico de Atividades:** Registro de notas e próximos passos do atendimento.
- [x] **Dashboard Basico:** Visão geral do volume de negociações.
